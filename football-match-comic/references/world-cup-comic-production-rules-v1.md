# 2026 美加墨世界杯纪实漫画制作规则 v1

## 作品定位

这是一套基于现实世界杯事件的纪实漫画连载。所有赛事、人物、比分、时间线、晋级结果、红黄牌、VAR、伤病、发布会发言、出征和抵达等事件，都应来自已经发生并可核验的公开资料。

漫画创作负责把现实事件转译成画面节奏、情绪张力和分镜叙事，不虚构比赛事实。

## 发布规格

```text
发布场景：社媒滑图阅读
页面比例：3:4 竖图
页面形式：完整漫画页，接近 PDF 漫画书单页质感
单集长度：1-5 页漫画为主
封面策略：普通事件不默认制作封面；重点章节、重大事件、合集或分卷可制作封面
默认风格：黑白日系足球纪实漫画
```

## 章节系统

```text
序章：世界杯之前
第一卷：小组赛风暴
第二卷：淘汰赛之门
第三卷：八强与四强
终卷：决赛与余音
特别篇：冷门、告别、争议、纪录
```

每个事件都归入一个章节。整部作品采用“全景群像 + 事件主角制”：世界杯整体是群像叙事，每一集根据现实事件确定临时主角。

## 事件工作流

```text
1. 输入事件点
2. 网络资料收集
3. 事实分层
4. 故事大纲提炼
5. 漫画页数判断
6. 分镜脚本设计
7. 图片 prompt 输出
8. 事实与漫画化边界说明
```

### 输入事件点

用户提供关键词、主题、阶段或具体事件，例如：

```text
世界杯开赛前，各个国家队开始出征
```

### 网络资料收集

根据事件点检索公开资料。优先使用 FIFA、赛事官方、国家队/足协官方、权威体育媒体、赛后发布会、球员公开社媒、现场图片或视频报道。

每个事件至少记录 2-3 个事实来源，方便后期回查。

### 事实分层

```text
确认事实：谁、何时、在哪里、发生了什么
背景信息：为什么重要，与世界杯进程有什么关系
漫画可用情绪：出征、告别、压力、荣耀、未知、爆发、退场等
```

### 故事大纲

把资料整理成可画的事件结构，而不是新闻流水账。大纲应提炼开场、推进、转折、爆点和余韵。

### 页数判断

```text
小事件：1 页
中事件：2-3 页
重大事件：4-5 页
超级事件：拆成上下集
```

单集不宜过长，避免社媒滑图阅读压力过高。

## 视觉圣经

### 页面规格

```text
比例：3:4 竖图
形式：完整漫画页
阅读：从上到下，优先左到右；如使用日漫式动线，需要用构图明确引导
每页格数：3-7 格
主格：每页至少 1 个大主格
```

### 分镜语言

- 避免平均九宫格、普通四格或整齐六格。
- 常用结构为大远景主格、窄条反应格、动作格、表情特写。
- 重大瞬间可以使用半页主格或跨页感主格。
- 每页至少包含两种镜头节奏：远景、近景、动作、静默。
- 禁止所有分格都横向堆叠。每页尽量包含至少一个竖向特写格或斜切动作格。

### 黑白质感

- 高对比黑白。
- 粗墨线外轮廓。
- 网点阴影。
- 速度线。
- 草皮线、汗水、尘土、灯光和看台声浪用漫画符号表现。
- 不使用彩色，不使用照片质感。

### 镜头库

```text
机场出征：登机口、行李箱、队服箱、球员背影、窗外航线
比赛现场：俯视跑位、脚下球、门将视角、球网、横梁、替补席
情绪特写：眼神、汗水、握拳、低头、背影、空座位
群像场景：看台、城市灯光、国旗海、记者席、球迷酒吧
```

### 角色画法

- 真实人物漫画化。
- 不追求照片级复刻。
- 保留身份线索：位置、发型轮廓、体态、球衣号码、球衣颜色转译。
- 脚本中可以写真实姓名；图片 prompt 中优先使用“角色特征 + 球衣颜色 + 位置”描述。

### 场景画法

- 球场要有空间感。
- 观众席可用密集黑白点阵表现。
- 城市、机场、更衣室、发布会、训练场都可以作为世界杯事件场景。
- 不画官方 logo、赞助商、水印。

### 文字系统

```text
语言：中文
文字量：每页主要文字 5-8 个信息点以内
标题：可使用竖排大字制造冲击
对白：短句为主
旁白：少量、纪实、克制
拟声：用于射门、奔跑、哨声、看台声浪
```

文字应该像漫画画面的一部分，而不是新闻说明书。

### 身份锚点

少放文字，但必须保留让读者进入现实事件的身份锚点。

```text
必须优先保留：球队名、关键人物名、关键分钟、比分/结果
可以保留：短旁白、短对白、拟声字
尽量减少：长解释、完整新闻句子、战术说明
```

每个文字块尽量短。优先使用标题牌、比分牌碎片、球衣背名、旁白框、人物名标签。涉及真实球员时，脚本里可以写真实姓名；图片 prompt 中可写人物名，但不要要求照片级复刻。

生图模型容易生成变形文字。正式发布时，建议让图像生成负责画面、分镜、留白和少量身份锚点，再由后期人工覆盖准确文字。

避免让模型随机生成球衣号码。除非号码已确认且确实需要出现在画面里，否则在 prompt 中加入：

```text
No jersey numbers unless explicitly requested.
Leave clean caption boxes for team names, player name, minute, and score.
```

## 分镜版式基准

默认不要使用全横向堆叠。用于日系足球漫画页的基准版式：

```text
上半页：一个大横向主格，占页面约 40-45%，展示球场空间、跑位、压迫或事件环境。
中段：一条窄横向反应格，占页面约 10-15%，展示队友、对手、教练、球迷或记者反应。
下半页左侧：一个大斜切动作格，占页面约 30-35%，展示突破、射门、扑救、登机、奔跑等关键动作。
下半页右侧：两个上下堆叠的竖向特写格，展示眼神、脚下球、球网、门柱、手套、护照、行李牌、比分牌等细节。
```

Prompt 中可明确写：

```text
Do not stack all panels as horizontal strips.
Do not make a regular grid.
Use varied panel shapes: one wide top panel, one thin reaction strip, one large diagonal action panel, two stacked vertical close-up panels.
```

### 禁止事项

- 不要彩色。
- 不要官方队徽。
- 不要赛事 logo。
- 不要赞助商。
- 不要长段解释文字。
- 不要把一页做成单张插画。
- 不要平均格子排版。
- 不要虚构比分、进球、红牌、VAR、晋级结果或赛后影响。

## 默认交付格式

每个事件默认输出：

```text
【事件点】
【所属章节】
【资料来源】
【事实资料包】
【故事大纲】
【漫画页数判断】
【分镜脚本】
【图片 Prompt】
【事实与漫画化边界说明】
```

## Prompt 基础风格片段

```text
black-and-white Japanese football documentary manga, complete 3:4 vertical manga page, non-uniform panel layout, one large main panel with smaller reaction and close-up panels, high contrast ink linework, screentone shadows, speed lines, expressive panel borders, clear reading order, cinematic football storytelling, Chinese short captions and speech bubbles only, no color, no official crests, no tournament logo, no sponsor marks, no watermarks
```

## Prompt 文字规则片段

```text
Chinese text only, large and legible. Include only these exact text items: "<team>", "<opponent>", "<player>", "<minute>", "<score>". No other text. Leave clean caption boxes if exact text placement is important. No jersey numbers unless explicitly requested.
```
